# Arquibancada CBLOW

Sejam muito bem vindos à arquibancada, este projeto é público e possui como objetivo entregar aos fãs do CBLOW uma interface onde eles possam interagir e imergir na caótica experiência que é o Campeonato Brasileiro de Low Elos.

A **Arquibancada CBLOW** não faz parte do projeto oficial, não somos filiados ao Bronziocre, Yoda ou qualquer outra instituição destes como a SL, este projeto é puramente feito para a comunidade e com objetivo de estudos e planejamento para campeonatos oficiais.

Canais dos hosts: [![Bronziocre na Twitch](https://img.shields.io/badge/Twitch-Bronziocre-9146FF?logo=twitch&logoColor=FFFFFF&style=for-the-badge)](https://www.twitch.tv/bronziocre) e [![Yoda na Kick](https://img.shields.io/badge/Kick-Yoda-53FC18?logo=kick&logoColor=000000&style=for-the-badge)](https://kick.com/yoda)

## Objetivo

Unificar informações em uma interface pública e interativa, perfeito para acompanhar durante as transmissões do Bronziocre e Yoda, ou das *watchparties* caso você prefira ;)

- histórico de todas as edições;
- histórico de partidas;
- chaveamento completo;
- banco de cortes;

Interessado em um jogador específico? Com a **Arquibancada** você poderá acompanhá-lo durante todo o campeonato.
